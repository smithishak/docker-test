# Простой блог на Node.js

## Запуск

1. Установите Node.js (если не установлен).
2. В терминале перейдите в папку проекта.
3. Запустите сервер командой:
   ```
   node server.js
   ```
4. Откройте браузер и перейдите по адресу http://localhost:3000

## Структура
- `server.js` — простой HTTP сервер
- `index.html` — главная страница блога
- `style.css` — стили для блога

## Описание
Это минимальный пример блога: две статьи, простая стилизация, без базы данных и сложной логики.
